## PCHelper — Phase 11g: External Device Manager (外部设备管理器)

### Overview
Add a comprehensive external device viewer: displays connected monitors (EDID info), audio devices, Bluetooth devices, printers, game controllers. Read-only status overview.

### Architecture

```
electron/
├── external/
│   └── deviceManager.ts  — All external device info

src/
├── components/
│   └── ExternalDevicesView/
│       ├── index.tsx            — Main view with 5 tabs
│       ├── DisplayInfo.tsx      — Monitor/EDID information
│       ├── AudioDevices.tsx     — Audio playback/recording devices
│       ├── BluetoothDevices.tsx — Paired BT devices
│       ├── Printers.tsx         — Installed printers
│       └── GameControllers.tsx  — Connected gamepads/joysticks
├── stores/
│   └── externalDevicesStore.ts
```

### Backend (`electron/external/deviceManager.ts`)

```typescript
export interface MonitorInfo {
  id: string;
  name: string;
  manufacturer: string;
  serialNumber: string;
  manufactureDate: string; // year/week
  screenSize: number; // inches
  resolution: string; // e.g. "1920x1080"
  refreshRate: number; // Hz
  connectionType: 'HDMI' | 'DP' | 'DVI' | 'VGA' | 'USB-C' | 'built-in';
  isPrimary: boolean;
  colorDepth: number; // bits
  aspectRatio: string;
  edidVersion: string;
  gpuOutput: string; // e.g. "NVIDIA GeForce RTX 3060 - HDMI"
}

export interface AudioDeviceInfo {
  id: string;
  name: string;
  type: 'playback' | 'recording' | 'both';
  deviceFormat: string; // e.g. "24bit, 48000Hz"
  isDefault: boolean;
  status: 'active' | 'disabled' | 'unplugged' | 'not_present';
  driver: string;
  jackInfo: string;
  volume: number; // 0-100
}

export interface BluetoothDeviceEntry {
  name: string;
  mac: string;
  type: 'keyboard' | 'mouse' | 'headset' | 'speaker' | 'phone' | 'watch' | 'other';
  paired: string; // date
  isConnected: boolean;
  batteryLevel: number | null;
  driver: string;
  signalStrength: number;
}

export interface PrinterEntry {
  name: string;
  driver: string;
  status: 'ready' | 'offline' | 'error' | 'paper_jam' | 'low_ink' | 'busy';
  isDefault: boolean;
  isNetwork: boolean;
  port: string;
  inkLevels?: Array<{ color: string; level: number }>; // 0-100
  pagesPrinted: number;
  lastUsed: string;
  location: string;
  supportsDuplex: boolean;
  supportsColor: boolean;
}

export interface GameControllerEntry {
  name: string;
  type: 'xbox' | 'playstation' | 'switch_pro' | 'generic' | 'joystick' | 'racing_wheel';
  connectionType: 'wired' | 'bluetooth' | 'wireless_dongle';
  isConnected: boolean;
  batteryLevel: number | null;
  buttons: number;
  axes: number;
  vendor: string;
  driver: string;
}

export class DeviceManager {
  getMonitors(): MonitorInfo[];
  getAudioDevices(): AudioDeviceInfo[];
  getBluetoothDevices(): BluetoothDeviceEntry[];
  getPrinters(): PrinterEntry[];
  getGameControllers(): GameControllerEntry[]
  refresh(): void;
}
```

Simulated data: 2 monitors (primary + secondary), 6-8 audio devices (speakers, headphones, mic, HDMI audio), 3-6 BT devices, 2-3 printers, 1-2 game controllers.

### Frontend (`src/components/ExternalDevicesView/`)

**Main (`index.tsx`):** 5-tab layout with icons

**DisplayInfo.tsx:**
- Cards for each monitor with EDID details
- Key info: name, size, resolution, refresh rate, connection type
- Primary monitor badge
- Technical details expandable: EDID version, serial, manufacture date, color depth

**AudioDevices.tsx:**
- Two sections: Playback devices (top) and Recording devices (bottom)
- Each device card: name, format, status badge, volume indicator
- Default device badge
- Driver info expandable

**BluetoothDevices.tsx:**
- Card layout per device: icon by type, name, connection status
- Battery level bar (if available)
- Signal strength indicator (bars)
- Paired date, driver info

**Printers.tsx:**
- Card per printer: name, status badge, default badge (star)
- Ink levels bar chart (if color)
- Details: port, pages printed, duplex/color support, location, last used

**GameControllers.tsx:**
- Card per controller: name, type icon, connection type badge
- Connection status with green/red indicator
- Battery level if wireless
- Technical info: buttons, axes, vendor, driver
