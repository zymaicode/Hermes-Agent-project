## PCHelper — Phase 11e: Local Group Policy Editor (本地策略编辑器)

### Overview
Add a read-only local group policy browser and search tool. Browse Windows policies by category, search for specific policies, view policy details and current status. Read-only — no modifications.

### Architecture

```
electron/
├── policy/
│   └── policyManager.ts  — Local group policy data provider

src/
├── components/
│   └── PolicyView/
│       ├── index.tsx        — Main view: sidebar categories + policy list + detail
│       ├── PolicyTree.tsx   — Category tree sidebar
│       ├── PolicyList.tsx   — Policy list with search/filter
│       └── PolicyDetail.tsx — Individual policy detail panel
├── stores/
│   └── policyStore.ts   — Zustand state
```

### Backend (`electron/policy/policyManager.ts`)

```typescript
export interface PolicyCategory {
  id: string;
  name: string;
  description: string;
  parentId: string | null;
  childCount: number;
  path: string; // e.g. "计算机配置 > 管理模板 > Windows 组件"
}

export interface PolicyEntry {
  id: string;
  categoryId: string;
  name: string;
  description: string;
  supportedOn: string; // e.g. "Windows 11, Windows 10"
  state: 'not_configured' | 'enabled' | 'disabled';
  helpText: string;
  registryKey: string; // READ-ONLY display — shows the key path for info
  registryValue: string;
  options?: string[]; // For policies with choices
}

export class PolicyManager {
  getCategories(): PolicyCategory[]
  getPoliciesByCategory(categoryId: string): PolicyEntry[]
  searchPolicies(query: string): PolicyEntry[]
  getPolicyById(id: string): PolicyEntry | null
}
```

### Frontend (`src/components/PolicyView/`)

**Main (`index.tsx`):** 3-panel layout:
- Left: PolicyTree (category sidebar)
- Center: PolicyList (policies in selected category)
- Right (or bottom panel): PolicyDetail (selected policy details)

**PolicyTree.tsx:**
- Hierarchical expandable tree of categories
- "Computer Configuration" and "User Configuration" root nodes
- Click to select → shows policies in center panel
- Search input at top

**PolicyList.tsx:**
- Table/grid of policies: name, state (badge: green=Enabled, red=Disabled, gray=Not Configured), description
- Sortable columns
- Each policy clickable → shows detail
- Filter: all / configured only / not configured
- Show count: "显示 24 条策略 (3 已配置)"

**PolicyDetail.tsx:**
- Full policy detail: name, description, supported OS, current state
- Registry key path display (read-only)
- Help text
- Supported options if applicable
- "📋 复制注册表路径" button
- Info badge: "此工具为只读查看，修改策略请使用 gpedit.msc"
