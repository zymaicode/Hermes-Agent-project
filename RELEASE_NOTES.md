# PCHelper v1.0-beta

## 功能概览
42 个系统工具，涵盖硬件监控、系统管理、网络诊断、安全防护、一键修复。

## 已完成功能 (42 项)
- **仪表盘** — 系统概览卡片，实时刷新硬件数据
- **硬件监控** — CPU/内存/磁盘/GPU 实时数据（WMI + 模拟回退）
- **软件管理** — 已安装应用列表、应用管理、更新检测
- **冲突检测** — 安装冲突、进程冲突分析
- **系统工具** — 进程管理、服务、驱动、开机启动、计划任务
- **网络工具** — 网络状态、连接列表、网络诊断（Ping/Traceroute/端口扫描/DNS/带宽测试）
- **安全工具** — 防火墙、安全中心、USB设备、用户账户管理
- **存储工具** — 磁盘清理、大文件扫描、重复文件查找
- **系统维护** — 系统信息、基准测试、系统还原、文件扫描、远程管理、硬件报告
- **性能工具** — 温度监控、电池报告、性能日志、内存分析
- **高级工具** — 注册表查看器、事件日志、文件类型、组策略查看器
- **驱动管理** — 驱动列表、备份、还原、版本对比
- **外部设备** — 显示器(EDID)、音频设备、蓝牙设备、打印机、游戏手柄
- **声音/字体** — 系统声音管理、字体管理
- **Windows 功能** — 功能开关管理
- **一键修复** — 7类问题检测、AI诊断、修复引擎（模拟阶段）
- **性能浮窗** — 可配置的 CPU/内存/GPU/FPS 悬浮窗
- **AI 助手** — DeepSeek AI 聊天面板

## 构建要求
- **Windows**: Node.js 22+, `npm run build` → dist/PCHelper Setup.exe
- **Linux**: Node.js 22+, `npm run build` → dist/PCHelper-*.AppImage

```bash
git clone ... && cd pc-toolkit
npm install
npm run build
```
