# PCHelper — 电脑工具 需求文档 v1.0

## 1. 项目概述

**项目名称：** 电脑工具（PCHelper）
**应用标识符：** `com.pchelper.app`
**开发框架：** Electron + React
**目标平台：** Windows（优先）→ 后续拓展 macOS
**界面风格：** 专业工具风，深色主题（类似 AIDA64）
**打包方式：** exe 安装包 + 便携版（免安装解压即用）

## 2. 功能模块

### MVP 第一期（核心四大模块）

#### 2.1 硬件检测模块
- **检测方案：** WMI/CIM + LibreHardwareMonitor 双方案互补
- **采集频率：** 持续实时，每秒刷新（类似任务管理器）
- **检测范围：**
  - CPU（型号、频率、核心数、温度、占用率）
  - 内存（容量、频率、类型、占用率）
  - 磁盘（型号、容量、接口类型、温度、读写速率、健康状态）
  - GPU（型号、显存、温度、占用率）
  - 主板（型号、芯片组、BIOS版本）
  - 网卡、声卡等外设信息

#### 2.2 软件检测模块
- **功能：** 获取系统所有已安装软件列表
- **信息：** 软件名称、版本、发布者、安装日期、安装路径
- **检测范围：** 系统所有已安装软件

#### 2.3 基础仪表盘
- **布局：** 仪表盘首页 + 详细页（概览卡片，点进去看详情）
- **内容：**
  - 硬件概览卡片（CPU/内存/磁盘/GPU 实时数据）
  - 实时数据刷新（每秒）
  - 卡片展示关键指标
- **AI入口：** 仪表盘上提供 AI 助手入口按钮

#### 2.4 AI 聊天面板
- **交互方式：** 侧边栏聊天式面板（类似 ChatGPT）
- **位置：** 嵌入应用右侧/侧边栏
- **功能：**
  - 用户输入问题，AI 回复
  - 可查询硬件状态、分析问题、给出建议
  - 支持上下文对话
- **AI模型：**
  - 默认：DeepSeek V4 Pro
  - 支持用户自定义（兼容 OpenAI 格式 API）
  - ⚠️ **模型优先级：DeepSeek V4 Pro 优先，V4 Flash 仅用于降级**

### 后续增加（MVP 第二期）

#### 2.5 冲突检测
- **检测范围：**
  - 安装冲突（同名软件、版本冲突）
  - 残留文件检测（已卸载但留下文件/文件夹）
  - 残留注册表检测（只读检测，报告用户手动清理）
  - 进程冲突（运行时冲突检测）

#### 2.6 应用管理
- 卸载入口
- 批量管理
- 应用详情查看

#### 2.7 软件更新检测
- 检测系统所有已安装软件的最新版本
- 提供更新建议

#### 2.8 AI 自动告警
- AI 自动分析实时数据
- 弹窗展示告警报告
- 本地规则快速告警 + AI 深度分析建议结合

#### 2.9 健康评分
- 综合硬件 + 软件状态评估
- 给出健康评分和优化建议

## 3. 技术栈

| 类别 | 选择 |
|------|------|
| **前端框架** | React (Vite) |
| **桌面框架** | Electron |
| **UI 组件库** | 待定（推荐 Material-UI 或 Ant Design，深色主题支持好） |
| **图表库** | Recharts 或 ECharts |
| **后端（硬件数据）** | Node.js + child_process（WMI）/ LibreHardwareMonitor 桥接 |
| **数据库** | SQLite |
| **AI API** | 兼容 OpenAI 格式，默认 DeepSeek V4 Pro |
| **打包** | electron-builder（exe 安装包 + 便携版） |

## 5. 数据流

```
系统硬件 (WMI/LibreHardwareMonitor)
  │
  ▼
Node.js 后端 (IPC 通信)
  │
  ▼
Electron 主进程 (数据处理 + SQLite 存储)
  │
  ▼
React 渲染进程 (UI 展示)
  │
  ▼
AI Chat Panel (DeepSeek V4 Pro API)
```

## 7. 关键决策记录

- **硬件检测：** WMI/CIM + LibreHardwareMonitor 双方案互补
- **数据采集频率：** 每秒刷新实时
- **AI 告警机制：** 本地规则快速告警 + AI 深度分析建议结合
- **界面风格：** 专业工具风，深色主题（AIDA64风格）
- **数据库：** SQLite
- **注册表策略：** 只读检测，报告用户手动清理，不自动清理
- **冲突检测范围：** 安装冲突 + 残留文件/注册表 + 进程冲突
- **软件更新检测：** 检测系统所有已安装软件
- **目标用户：** 面向公众使用
- **AI 模型：** DeepSeek V4 Pro 默认，支持自定义（兼容 OpenAI 格式）
- **打包方式：** exe 安装包 + 便携版（两者都提供）
- **AI 交互：** 聊天面板 + 自动告警弹窗（两者都有）
- **布局风格：** 仪表盘首页 + 详细页
- **MVP分期：** 第一期核心4项（硬件检测、软件检测、仪表盘、AI聊天面板）
