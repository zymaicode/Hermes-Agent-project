## PCHelper v1.0-beta 发布计划

### 目标
构建可发布的 Windows 安装包，让用户在真机上真正运行 PCHelper。

> **执行方式：** 每个步骤自行通过 Claude Code 或 delegate_task 执行

---

### Step 1: 修复构建问题
**Objective:** 准备构建资产（icon）并让 `npm run build` 在 Linux 上能够通过（至少到 Vite build 阶段）

**Files:**
- Create: `assets/icon.png` (Placeholder — 1x1 transparent PNG, 256x256)
- Modify: (none needed)

**Verification:**

**Objective:** 让 tempCleaner、browserCleaner、systemCleaner 在 Windows 上真正执行清理操作

**Files:**
- Read: `electron/cleanup/tempCleaner.ts`
- Read: `electron/cleanup/browserCleaner.ts`
- Read: `electron/cleanup/systemCleaner.ts`

**Goal:** 在代码逻辑层加上 platform 检测 + 真实 Windows 命令执行（`del /q/f/s %TEMP%` 等），但保留模拟回退。

**Verification:** tsc 编译通过

### Step 3: 连接 Overlay 到真实硬件数据

**Objective:** 让 Overlay 数据采集器使用 HardwareCollector 的真实数据而非纯模拟

**Files:**
- Read: `electron/overlay/dataCollector.ts`

**Goal:** 在 dataCollector 中 import HardwareCollector，获取真实的 CPU/内存/GPU 数据，只模拟 FPS 和网络。

**Verification:** tsc 编译通过
