## 步骤 1: 构建资产与验证

**Objective:** 创建 icon 占位文件，验证 Vite build 通过

**Files:**
- Create: `assets/icon.png`

**Steps:**

1. 用 Python 创建 256x256 placeholder PNG:

```python
# Create minimal valid PNG (256x256, dark blue)
import struct, zlib

def create_png(width, height, r, g, b):
    def chunk(type, data):
        c = type + data
        return struct.pack('>I', len(data)) + c + struct.pack('>I', zlib.crc32(c) & 0xffffffff)
    
    raw = b''
    for y in range(height):
        raw += b'\x00'  # filter byte
        for x in range(width):
            raw += bytes([r, g, b, 255])  # RGBA
    
    ihdr = struct.pack('>IIBBBBB', width, height, 8, 6, 0, 0, 0)  # 8-bit RGBA
    
    return (b'\x89PNG\r\n\x1a\n' +
            chunk(b'IHDR', ihdr) +
            chunk(b'IDAT', zlib.compress(raw)) +
            chunk(b'IEND', b''))

with open('/home/ubuntu/workspace/pc-toolkit/assets/icon.png', 'wb') as f:
    f.write(create_png(256, 256, 30, 100, 200))
```

2. 运行 Vite build 验证:
```bash
cd /home/ubuntu/workspace/pc-toolkit
npx vite build 2>&1 | tail -10
```

3. 尝试完整 build:
```bash

```
